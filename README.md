## ✨ Temel Özellikler

- **4 Modern Tema** (Koyu/Açık/Gruvbox/Vampire)

- **Çoklu Dil Desteği**: TR/AR/EN/DE/FR
- **Kıble Yön Bulucu** (Pusula ve açı)
